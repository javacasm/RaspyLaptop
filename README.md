* Raspberry
* Arduino leonardo (para el Trackpad y la bateria)

* Miniteclado usb
* Pantalla de 7"
* Pantalla oled (para el nivel de batería)
* hub usb
* pantalla oled (para la batería)
* Circuito de powerbank ¿y si usamo uno con lcd?
* 3 baterías 18650
